  # 操作系统详细知识图谱

## 1. 操作系统概述

### 1.1 基本概念
- **定义**: 管理计算机硬件与软件资源的系统软件
- **目标**: 
  - 方便性: 使计算机更易使用
  - 有效性: 提高系统资源利用率
  - 可扩展性: 支持系统功能扩展
  - 开放性: 支持标准接口
- **功能**:
  - 处理器管理
  - 存储器管理
  - 设备管理
  - 文件管理
  - 作业管理

### 1.2 操作系统分类
- **按使用环境分类**:
  - 桌面操作系统 (Windows, macOS, Linux)
  - 服务器操作系统 (Linux, Windows Server)
  - 移动操作系统 (Android, iOS)
  - 嵌入式操作系统 (FreeRTOS, VxWorks)
- **按处理方式分类**:
  - 批处理系统
  - 分时系统
  - 实时系统
  - 分布式系统
- **按支持用户数分类**:
  - 单用户系统
  - 多用户系统

### 1.3 操作系统结构
- **简单结构** (MS-DOS)
- **分层结构** (THE系统)
- **微内核结构** (Mach)
- **模块化结构** (现代Linux)
- **混合结构** (Windows NT)

## 2. 进程管理

### 2.1 进程基本概念
- **进程定义**: 程序在执行过程中的一个实例
- **进程特征**:
  - 动态性: 动态创建、执行、消亡
  - 并发性: 多个进程可并发执行
  - 独立性: 进程间相互独立
  - 异步性: 进程执行速度不可预知
- **进程组成**:
  - 程序代码 (文本段)
  - 数据段
  - 堆栈段
  - 进程控制块 (PCB)

### 2.2 进程状态与转换
- **三态模型**:
  - 运行态 (Running): 正在CPU上执行
  - 就绪态 (Ready): 等待CPU分配
  - 阻塞态 (Blocked): 等待某个事件发生
- **五态模型**:
  - 新建态 (New): 进程正在创建
  - 就绪态 (Ready)
  - 运行态 (Running)
  - 阻塞态 (Blocked)
  - 终止态 (Terminated): 进程执行完毕
- **七态模型**:
  - 增加就绪挂起态和阻塞挂起态

### 2.3 进程控制块 (PCB)
- **基本信息**:
  - 进程标识符 (PID)
  - 进程名称
  - 用户标识符
- **处理器状态**:
  - 通用寄存器
  - 指令计数器
  - 程序状态字
  - 用户栈指针
- **进程控制信息**:
  - 调度和状态信息
  - 进程间通信信息
  - 存储管理信息
  - 资源占用信息

### 2.4 进程调度
- **调度层次**:
  - 高级调度 (作业调度)
  - 中级调度 (内存调度)
  - 低级调度 (进程调度)
- **调度算法**:
  - **先来先服务 (FCFS)**:
    - 优点: 简单、公平
    - 缺点: 平均等待时间长
  - **短作业优先 (SJF)**:
    - 优点: 平均等待时间最短
    - 缺点: 可能出现饥饿现象
  - **优先级调度**:
    - 静态优先级
    - 动态优先级
  - **时间片轮转 (RR)**:
    - 适用于分时系统
    - 时间片大小影响性能
  - **多级反馈队列**:
    - 结合多种算法优点
    - 适应性强

### 2.5 进程同步与通信
- **同步机制**:
  - **信号量 (Semaphore)**:
    - 二元信号量
    - 计数信号量
    - P操作和V操作
  - **管程 (Monitor)**:
    - 条件变量
    - 等待队列
  - **互斥锁 (Mutex)**:
    - 临界区保护
    - 死锁预防
- **通信机制**:
  - **共享内存**:
    - 高效通信方式
    - 需要同步控制
  - **消息传递**:
    - 直接通信
    - 间接通信
  - **管道 (Pipe)**:
    - 匿名管道
    - 命名管道 (FIFO)

### 2.6 死锁问题
- **死锁定义**: 两个或多个进程无限期地等待永远不会发生的事件
- **必要条件**:
  - 互斥条件
  - 请求和保持条件
  - 不剥夺条件
  - 环路等待条件
- **处理策略**:
  - **预防死锁**: 破坏必要条件之一
  - **避免死锁**: 银行家算法
  - **检测死锁**: 资源分配图
  - **解除死锁**: 进程终止、资源抢占

## 3. 内存管理

### 3.1 内存管理基础
- **内存管理功能**:
  - 内存分配与回收
  - 地址变换
  - 内存保护
  - 内存扩充
- **程序装入方式**:
  - 绝对装入
  - 可重定位装入
  - 动态运行时装入
- **地址类型**:
  - 逻辑地址
  - 线性地址
  - 物理地址

### 3.2 连续内存分配
- **单一连续分配**:
  - 用户区和系统区
  - 简单但利用率低
- **固定分区分配**:
  - 等长分区
  - 不等长分区
  - 内部碎片问题
- **动态分区分配**:
  - 首次适应算法
  - 最佳适应算法
  - 最坏适应算法
  - 快速适应算法
  - 外部碎片问题

### 3.3 虚拟内存管理
- **虚拟内存概念**:
  - 逻辑上扩大内存容量
  - 部分装入、部分对换
  - 局部性原理支持
- **页式存储管理**:
  - **基本原理**:
    - 页面和页框
    - 页表映射
    - 地址变换
  - **页表结构**:
    - 单级页表
    - 多级页表
    - 反向页表
  - **快表 (TLB)**:
    - 地址变换加速
    - 硬件支持
- **段式存储管理**:
  - 段表
  - 段内地址
  - 段间共享
- **段页式存储管理**:
  - 段表 + 页表
  - 两次地址变换

### 3.4 页面置换算法
- **最佳置换算法 (OPT)**:
  - 理论最优
  - 实际无法实现
- **先进先出算法 (FIFO)**:
  - 简单易实现
  - 可能出现Belady异常
- **最近最久未使用算法 (LRU)**:
  - 性能较好
  - 实现复杂
- **时钟算法 (Clock)**:
  - LRU的近似实现
  - 使用访问位
- **最不常用算法 (LFU)**:
  - 统计使用频率
  - 开销较大

### 3.5 内存分配策略
- **固定分配局部置换**
- **可变分配全局置换**
- **可变分配局部置换**
- **工作集模型**:
  - 工作集窗口
  - 工作集大小
  - 缺页率控制

## 4. 文件系统

### 4.1 文件系统概述
- **文件概念**:
  - 文件是具有名称的相关信息的集合
  - 文件属性: 名称、标识符、类型、位置、大小、保护、时间和日期
- **文件类型**:
  - 普通文件
  - 目录文件
  - 特殊文件 (设备文件)
- **文件系统功能**:
  - 文件存储空间管理
  - 目录管理
  - 文件读写管理
  - 文件共享与保护

### 4.2 文件组织结构
- **文件逻辑结构**:
  - **无结构文件** (流式文件)
  - **有结构文件** (记录式文件):
    - 顺序文件
    - 索引文件
    - 索引顺序文件
    - 直接文件 (散列文件)
- **文件物理结构**:
  - **连续分配**:
    - 优点: 支持顺序访问和随机访问
    - 缺点: 外部碎片、文件大小难以确定
  - **链接分配**:
    - 隐式链接
    - 显式链接 (FAT)
  - **索引分配**:
    - 单级索引
    - 多级索引
    - 混合索引

### 4.3 目录管理
- **目录结构**:
  - **单级目录**: 简单但不支持重名
  - **二级目录**: 支持用户独立空间
  - **树形目录**: 层次结构，支持路径
  - **图形目录**: 支持链接，可能产生环
- **目录操作**:
  - 搜索文件
  - 创建文件
  - 删除文件
  - 显示目录
  - 修改目录

### 4.4 文件存储空间管理
- **空闲块管理**:
  - **空闲表法**: 记录空闲区域
  - **空闲链表法**: 链接空闲块
  - **位图法**: 使用位图标记
  - **成组链接法**: 分组管理空闲块
- **文件分配方法**:
  - 连续分配
  - 链接分配
  - 索引分配

### 4.5 文件系统实现
- **文件系统结构**:
  - 引导块
  - 超级块
  - 空闲块信息
  - i-node区
  - 数据区
- **文件控制块 (FCB)**:
  - 文件基本信息
  - 存取控制信息
  - 使用信息
- **文件描述符表**:
  - 进程级文件描述符表
  - 系统级文件描述符表

## 5. 输入/输出系统

### 5.1 I/O系统概述
- **I/O设备分类**:
  - **按使用特性**:
    - 存储设备
    - 输入输出设备
    - 网络通信设备
  - **按传输速率**:
    - 低速设备
    - 中速设备
    - 高速设备
  - **按信息交换单位**:
    - 块设备
    - 字符设备
- **I/O控制方式**:
  - 程序直接控制
  - 中断驱动
  - DMA方式
  - 通道控制方式

### 5.2 设备管理
- **设备分配**:
  - 设备独占性
  - 设备分配算法
  - 设备分配安全性
- **设备驱动程序**:
  - 设备驱动程序接口
  - 设备驱动程序结构
  - 设备中断处理
- **设备无关性**:
  - 逻辑设备名
  - 逻辑设备表
  - 设备驱动程序

### 5.3 缓冲管理
- **缓冲的引入**:
  - 缓解CPU与I/O设备间速度不匹配
  - 减少对CPU的中断频率
  - 提高CPU与I/O设备的并行性
- **缓冲类型**:
  - **单缓冲**: 一个缓冲区
  - **双缓冲**: 两个缓冲区轮换
  - **循环缓冲**: 多个缓冲区循环使用
  - **缓冲池**: 公共缓冲区集合

### 5.4 磁盘调度算法
- **先来先服务 (FCFS)**:
  - 公平但效率低
- **最短寻道时间优先 (SSTF)**:
  - 效率高但可能饥饿
- **扫描算法 (SCAN)**:
  - 电梯算法
  - 避免饥饿
- **循环扫描算法 (C-SCAN)**:
  - 单向扫描
  - 等待时间更均匀
- **N-Step-SCAN算法**:
  - 分段扫描
- **FSCAN算法**:
  - 双队列扫描
